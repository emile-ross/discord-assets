# discord assets

some assets I use for discord.

This is compatible with the kettu mobile discord client.


# Install Themes

Paste one of the following links in the Theme installation box (in the kettu client)

### Midnight Vencord theme link:

```
https://raw.githubusercontent.com/emile-ross/discord-assets/main/kettu/theme.json
```

### Skymoled theme link:

```
https://raw.githubusercontent.com/emile-ross/discord-assets/main/kettu/skymoled.json
```

[source](https://github.com/kmmiio99o/discord-mobile-themes/blob/main/midnight-vencord.json)


# Install Fonts

Paste one of the following links in the Font installation box (in the kettu client)

### JetBrainsMono font link:
```
https://raw.githubusercontent.com/emile-ross/discord-assets/main/fonts_kettu/JetBrainsMono/jetbrains.json
```

### JetBrainsMonoNL font link:

This is a version of the JetbrainMono font but without ligatures 
(when two letters like a and e "ae" are joined together to form a single glyph)

```
https://raw.githubusercontent.com/emile-ross/discord-assets/main/fonts_kettu/JetBrainsMonoNL/jetbrains-nl.json
```
